🔍 Project Explanation: HR Data Analysis using Power BI

This project focuses on analyzing Human Resources (HR) data to help organizations understand their workforce better and take data-driven HR decisions.

📌 What problem does this project solve?

HR teams often have raw employee data but no clear insights. This project converts that raw data into interactive dashboards that answer key questions like:

How many employees do we have?

Which departments have the highest attrition?

What is the age and gender distribution?

Where should management focus to reduce employee turnover?

📊 Data Used

The dataset contains employee-level information such as:

Employee ID

Age and Gender

Department and Job Role

Monthly Income

Years of Experience

Performance Rating

Attrition status (Yes/No)

This data represents a typical corporate HR system.

🔄 Data Preparation (ETL Process)

I used Power Query in Power BI to:

Remove duplicate and null values

Clean and standardize columns

Convert data types (text, number, date)

Prepare the dataset for accurate analysis

This ensures clean and reliable data before visualization.

🧮 Data Modeling & DAX

I created calculated measures using DAX, such as:

Total Employees

Attrition Count

Attrition Rate (%)

Average Age

Average Monthly Income

These KPIs help measure overall workforce health.

📈 Dashboards & Visuals

The project includes multiple interactive dashboards:

1️⃣ Employee Overview

Total employees

Active vs attrition employees

Department-wise headcount

2️⃣ Attrition Analysis

Attrition by department

Attrition by job role

Attrition by age group

This helps identify problem areas with high employee turnover.

3️⃣ Demographics Analysis

Gender distribution

Age-wise employee segmentation

Experience-based analysis

🎯 Key Insights

Certain departments show higher attrition rates

Younger age groups have more frequent job changes

Some roles have lower retention compared to others

Salary and experience impact employee stability

💼 Business Value

This dashboard helps:

HR managers reduce attrition

Management plan better hiring strategies

Identify departments needing engagement programs

Improve workforce planning and retention policies
